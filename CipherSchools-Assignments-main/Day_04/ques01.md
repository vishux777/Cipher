### Question-01

Write a C++ program that takes two integers as input and performs the following arithmetic operations:
Addition
Subtraction
Multiplication
Division (ensure that you handle division by zero)
Modulus

**Sample Input:**

```
Enter first integer: 10
Enter second integer: 3
```

**Sample Output:**

```
Addition: 13
Subtraction: 7
Multiplication: 30
Division: 3.33333
Modulus: 1
```
